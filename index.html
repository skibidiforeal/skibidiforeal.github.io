Aurum Language Documentation

About Aurum

Aurum is an all-level programming language designed to simplify coding and allow developers to work with low-level, mid-level, and high-level constructs all in one language. It is built with ease of use in mind, with a syntax based on C#, but optimized for various levels of abstraction, making it flexible for both beginners and advanced programmers.

Currently, the high-level features of Aurum are available and functional, with mid-level and low-level features being developed. The language aims to support everything from basic scripting to deeper system-level programming, enabling developers to write optimized code without switching between multiple languages.

Introduction

The Aurum interpreter is a simplified C#-based interpreter that supports basic programming constructs like arithmetic expressions, variable declarations, and conditional statements. It also features integration with OpenAI's GPT-3 model to generate code dynamically based on user-provided queries.

Key Components

The TokenType enum represents different types of tokens in the language, such as numbers, arithmetic operators, identifiers, and AI-related commands.

public enum TokenType { NUMBER, PLUS, MINUS, STAR, SLASH, ASSIGN, SEMICOLON, LEFT_BRACE, RIGHT_BRACE, IF, PRINT, IDENTIFIER, INT, EOF, AI }

public class Token
{
    public TokenType Type { get; }
    public string Lexeme { get; }

    public Token(TokenType type, string lexeme)
    {
        Type = type;
        Lexeme = lexeme;
    }

    public override string ToString()
    {
        return $"{Type} -> {Lexeme}";
    }
}
