Introduction

The Aurum interpreter is a simplified C#-based interpreter that supports basic programming constructs like arithmetic expressions, variable declarations, and conditional statements. It also features integration with OpenAI's GPT-3 model to generate code dynamically based on user-provided queries.

Key Components

1. Token and TokenType Classes

The TokenType enum represents different types of tokens in the language, such as numbers, arithmetic operators, identifiers, and AI-related commands. The Token class encapsulates the token type and the corresponding lexeme (text).

public enum TokenType { NUMBER, PLUS, MINUS, STAR, SLASH, ASSIGN, SEMICOLON, LEFT_BRACE, RIGHT_BRACE, IF, PRINT, IDENTIFIER, INT, EOF, AI }

public class Token
{
    public TokenType Type { get; }
    public string Lexeme { get; }

    public Token(TokenType type, string lexeme)
    {
        Type = type;
        Lexeme = lexeme;
    }

    public override string ToString()
    {
        return $"{Type} -> {Lexeme}";
    }
}

Aurum Syntax

Aurum is designed to be a simple and easy-to-understand language, similar to C# but with a focus on simplicity. Below are examples of syntax in Aurum, along with their purposes:

1. Variable Declaration

int x = 5;

Declares an integer variable x and assigns it the value of 5.

2. Printing Output

print x;

Prints the value of the variable x.

3. Conditional Statements

if (x > 3) {
    print "x is greater than 3";
}

Executes the print statement if x is greater than 3.

4. Arithmetic Operations

int result = x + 2 * 3;

Performs arithmetic operations, following the order of operations.

5. While Loop

while (x < 10) {
    x = x + 1;
}

Increments x until it reaches 10.

6. Function Declaration

function Add(int a, int b) {
    return a + b;
}

Declares a function named Add that returns the sum of two integers.

7. AI Query

ai = (Generate a function that prints Fibonacci numbers)

Generates code using OpenAI's API based on the userâ€™s query.

How to Use Aurum

Aurum can be used to quickly prototype programs, perform basic computations, and even integrate with OpenAI for generating code dynamically. Below are steps on how to use the Aurum interpreter.

Step 1: Write Code

Write simple Aurum code directly in the command line of the interpreter:

int x = 5;
print x;

Step 2: Execute Code

Simply run the interpreter and type the code. The interpreter will evaluate and execute the commands. You can print values, declare variables, and use control flow constructs like if statements and loops.

Step 3: AI Integration

To use the AI integration, use the ai = (query) command. For example, ask the AI to generate code:

ai = (Generate a function to sort an array)

This will query OpenAI and return the generated code to the user, which you can then execute in the interpreter.

C# Integration for Modding

Aurum allows integration with C# games, making it easier to create mods for games built with Unity or other C# engines. Below is a basic example of how to integrate Aurum into C# code to extend or modify a Unity game.

Unity Modding Example (C#)

This example shows how to hook into a Unity game and allow Aurum scripts to modify certain behaviors. You can use this as a base to add custom functionality to any Unity-based game using Aurum.

// Example Unity mod to integrate Aurum
using UnityEngine;

public class AurumMod : MonoBehaviour
{
    void Start()
    {
        Debug.Log("Aurum Mod Loaded");
        // Example: Add a key to run an Aurum script
        RunAurumScript();
    }

    void RunAurumScript()
    {
        // This is where you can integrate the Aurum interpreter
        // and load a script written in Aurum to modify game behavior.
        string aurumScript = @"
            int health = 100;
            health = health - 25;
            print health;
        ";

        AurumInterpreter interpreter = new AurumInterpreter();
        interpreter.Run(aurumScript);
    }
}
